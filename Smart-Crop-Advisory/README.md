# 🌱 Smart Crop Advisory System

### 🚀 AI-Powered Farming Assistance Platform

> Empowering farmers with smart agricultural insights 🌾

---

## 📌 Overview

Smart Crop Advisory System is a modern AgriTech web application that helps farmers make better decisions using technology.

---

## ✨ Features

* 🌱 Crop Recommendation
* 🌦 Weather Advisory
* 💰 Market Prices
* 🧪 Pesticide Calculator
* 📅 Farming Schedule
* 🤖 AI Chatbot
* 🌍 Hindi + English Support
* ♿ Accessibility Features

---

## 🏗️ Tech Stack

Frontend:

* React (Vite)
* Tailwind CSS
* Framer Motion
* Recharts

Backend:

* Node.js / Spring Boot

Database:

* MongoDB / MySQL

---

## 🚀 Run Project

```bash
npm install
npm run dev
```

App runs at:
http://localhost:5173
